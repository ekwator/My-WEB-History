Node Gallery

Модуль Node Gallery позволяет пользователям создавать галереи. Управляющий сайтом может определить тип документов, который будет использоваться для создания галереи и тип документов, который будет использоваться для добавления в галерею отдельных изображений.

Так как и галереи и изображения в них являются документами, то они могут использовать все преимущества Друпала по работе с документами и работать совместно с модулями Taxonomy, Comment, Voting API, CCK, Views, Nodeaccess и любыми другими модулями. Всё, что можно делать с документами, можно делать и с галереями и с изображениями.

Возможности модуля

  • Использование стандартного Batch API для выполнения пакетных операций (таких как загрузка файлов и их правка)
  • Создание неограниченного количества галереи и изображений в них
  • Создание пользователями своих собственных галерей
  • Интеграция с модулем Token
  • Интеграция с модулем Lightbox2 (который позволяет использовать различные эффекты при показе изображений или создавать слайдшоу)

Установка модуля

При включении, модуль Node Gallery автоматически создаст 2 новых типа документов (один для галерей, другой для изображений галерей), 3 новых набора правил обработки изображений для ImageCache (для изображения на обложке, миниатюр и стандартных изображений) и создаст связи для создания галерей (связав созданные типы документов).

Дополнительные модули

Для модуля Node Gallery существуют различные модули, которые позволяют расширить его возможности.

  • Node Gallery Access
  • Node Gallery jCarousel
  • Node Gallery Slideshow
  • Node Gallery Taxonomy
  • Node Gallery User Profile
