Главная

File management
filehash

Генерирует и сохраняет MD5, SHA-1 и / или SHA-256 хэши для каждого загруженного на сайт файла.

filemanager

API для управления файлами на сайте.

filemime

Указание MIME типов файлов.

filenode

Создаёт материал при загрузке файла.

filepicker

Инструментом, чтобы помочь загружать, просматривать и добавлять ссылки на файлы в материалах.

filerelationsserver

Доступ к разным типам MIME файлов.

fileserver

Добавляет DAV API для доступа к файлам.

files_proxy

Позволяет использовать CDN в мультисайтинге.

filetree

Входной фильтр, который предоставляет список файлов в красивом, интерактивном дереве файлов. Использует (<2 кб) Javascript файл для показа разворачивающихся и сворачивающихся папок, с иконками отображающими различные типы файлов.

fileview

Позволяет загружать файлы на сайт.

fileviewer

Добавляет Internet Archive BookReader для PDF и EPUB поддержку в Drupal, чтобы их можно открыть и просмотреть в браузере.

file_access

Предназначен для запрета или разрешения отдельным файлам доступа в Drupal.

file_access_by_node_type

Позволяет настраивать доступ к файлам для выбранных типов материалов.

file_aliases

Модуль расширяет стандартные возможности FileField.

file_defer

Ищет потерянные файлы.

file_download_form

Позволяет администраторам задавать пользователю условие указания адреса электронной почты для того, чтобы скачать файл, загруженный с помощью filefield.

file_entity

Расширяет сущность полей, позволяя полям иметь формат отображения и др.

file_force

Вынуждает скачать файл, вместо его показа (как например PDF).

file_maintenance

Файловый менеджер на сайте.

file_newest_revision

Позволяет сохранять постоянные ссылки к изменяемым файлам.

flashvideo

Комплексное решение для работы с video.

flickrinsert

Позволяет включить полные Flickr серии в содержание вашего сайта, будь то материалы, блоки и даже комментарии.

gallery_assist_plupload

Расширяет галерею Gallery Assist возможностью множественной загрузки через Plupload.

hash_wrapper

Оболочка для Drupal 7 реализации 'hash://' схемы хранения пару (десять) миллионов файлов.

hook_file

Это экспериментальный порт Drupal 7 в file.inc "hook_file" API для Drupal 6.

hotfolder

Предназначен для оказания помощи с автоматизированной обработкой данных. Отслеживает папку на сервере во время cron, и запускает на обработку если в папке будут обнаружены обновления.

httpHeaders

Для каждого ContentType администратор может определить, каким Last-Modified заголовки будут от по отношению к дате создания страницы.

ie6update

Позволяет побуждать пользователей обновить IE6 и ссылка на страницу обновлений.

imageblock

Позволяет легкую вставку изображений в блок.

imagebrowser

Браузер изображений для Drupal и FCKeditor.

imagecache_subdomain

Расширение для модуля ImageCache позволяющее использовать субдомены при сохранении изображений, формируемых ImageCache.

imagefield

Загрузка изображений для CCK

imagefield_archive

Добавляет новою операцию которая интегрируется с Views Bulk Operations и разрешает загрузку Zip архивов ImageFields содержащиеся в отдельных материалах.

imagefield_focus

Позволяет масштабировать и обрезать изображения с упором на их важные части. Очень удобно при использовании в ImageField и ImageCache.

imagefield_import

Позволит пользователям иметь возможность импортировать большое количество картинок в CCK, который содержатся в imagefield.

imagemagick

Позволяет использовать ImageMagick как API инструментарий для работы с изображениями в Drupal 7.

image_field_url_replacer

Позволяет заменить в SRC атрибут URL адрес изображения, загруженного в поля изображения.

image_fupload

Позволяет пользователям загружать несколько изображений.

imce

Модуль для загрузки изображений на сервер(использует роли, квоты, ресайзинг).

imceimage

Модуль для добавления полей с изображениями в CCK
