Главная

File management
delicious_cloud

Облако тегов для входа пользователей использует модуль delicious.

disknode

Создаёт материал предназначенный для хранения файлов.

docs

Представляет собой простой документо/файло менеджер.

downld

Добавляет возможность публичного и закрытого скачивания файлов.

download_count

Статистика скачивания для прикреплённых файлов.

dragndrop_uploads

Добавляет возможность перетащить изображение со своей локальной файловой системы, поместив его в материал Textarea в тело и файл автоматически загрузится в материал.

dragon

Автоматически преобразует все формы материала, с тем чтобы работала HTML5 функция перетаскивания файла File API.

drd_server

Серверный компонент модуля администратора DRD.

dropbox

Позволяет пользователям иметь страницу "Отправить файл", где пользователи могут отправлять файлы непосредственно с их Dropbox аккаунта.

drupal_ftp

Реализация FTP.

drush_make

Предоставляет эквивалент создания скрипта, используя Drush.

encl_remote

Позволяет пользователям добавлять отдельные ресурсы через RSS в материалы.

epsacrop

Позволяет менять координаты отображаемого изображения.

evernote

Добавляет интеграцию с Evernote (Evernote отлично подходит для создания заметок, сохранения интересных веб-страниц, голосовых сообщений, фотографий визитных карточек, для ведения списков дел и еще для многих других задач).

fancy_insert

обеспечивает более удобный интерфейс для вставки изображения в конкретных позиций в рамках содержимое узла..

fast_gallery

Галерея с множеством дополнительных функций.

fbsmp

Позволяет пользователям публиковать контент (ссылки, изображения и видео) в потоке, а также на их Facebook-style Statuses (как на Facebook).

feedapi_grabber

Является FeedAPI аддоном, который добавляет возможности вывода прикрепленные файлов в ленты RSS.

feeds_imagegrabber

Преемник FeedAPI ImageGrabber, с поддержкой Feeds модуля.

feed_tag_filter

Плагин к FeedAPI работающий со словарями.

filebanlist

Список запрещеных для загрузки типов файлов.

filebrowser

Предоставляет доступ для работы как FTP-интерфейс сайта.

filedepot

Полнофункциональный модуль управления документооборотом, как Google Docs.

filefield

Обеспечивает загрузку для CCK.

filefield_audio_insert

Расширяет функциональность модуля FileField CCK, позволяя вставить воспроизводимый через Flash звук в материал. Работает в сочетании с mp3player модулем для воспроизведения звука.

filefield_dl

Добавляет возможность начала автоматической закачки файла при просмотре страниц.

filefield_insert

Позволяет добавить изображения и ссылки на файлы в текстовых областях или WYSIWYGs гораздо проще. Он добавляет, простой JavaScript к кнопке FileField и ImageField. При использовании с ImageField и ImageCache, изображения могут быть вставлены в текст с ImageCache предустановки.

filefield_paths

Позволяет добавлять пути к файлам.

filefield_private

Приватный метод загрузки файлов.

filefield_sources

Модуль позволяет загружать файлы с вашего компьютера через CCK поля.

filefield_sources_node_title

Расширение к Filefield Sources, позволяющее найти файлы, основанные на Title материала.

filefield_sources_noupload

В некоторых случаях необходимо запретить виджет загрузки — например, вынудить пользователей загрузить использование определенного типа контента и только справочных файлов для различного типа контента.

filefield_sources_reference_opt

Расширение к FileField Sources модулю, добавляющее описание и автозаполнение.

filefield_sources_taxonomy

Расширение к Filefield Sources, позволяющее найти файлы, основанные на терминах словаря.

filefield_stats

Позволяет отслеживать статистику скачивания в CCK.

filefield_styles

Совместно с модулем Styles позволяет использовать в Filefield поля разного типа.

filefield_track

Отслеживает CCK FileField загрузки и разрешает загрузки анонимным пользователям, если они введите адрес электронной почты.

filefield_tracker

Позволяет отслеживать загрузку filefields.

filefield_uiextras

Обеспечивает сбор различного усовершенствования пользовательского интерфейса для FileField.

fileframework

Загрузка в RSS поля данных из разных источников.
